# FantasyDreamStoryClubJAVA

绮梦故事会网站——后端


# 使用方式：
本项目为springboot项目，项目成品可通过访问域名查看：www.qmstory.club
项目启动前请修改application.yml文件来配置数据库、redis、端口图片存放地址等
该项目为开源项目，欢迎大家提出修改意见或学习参考。
